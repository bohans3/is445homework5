Scatter Plot

The data I used comes from a dataset of Bigfoot reports in the United States. My first visualization is a scatter plot that displays the geographical distribution of these reports based on latitude and longitude. Each point on the plot represents an individual report, offering a spatial perspective on where sightings have occurred. I used the X and Y axes, color encoding, and tooltip encoding to enhance the visualization. The points are color-coded based on the state in which each sighting was reported. Additionally, I incorporated interactive features such as pan/zoom and multiple selection tools. For data transformation, I used pandas to load the CSV file and added tooltips to display additional information when viewers hover over each point.
